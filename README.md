# Tap Click Counter

A tiny Windows desktop app: a counter you increase by clicking `+`, the
window background, or decrease with `-` (floors at 0). The pin button in
the top-right toggles the window as always-on-top, like Windows 11
Calculator's "Keep on top."

## Build

Requires `gcc` (mingw-w64) on your `PATH`.

```
build.bat
```

Produces `counter.exe` (no other files or runtime dependencies needed).

## Run

```
counter.exe
```
